MI Services

Welcome to the MI services page.

Here, you can access all the web services provided by MI that can't be shown on a static web page!

Note: I currently provide all services from my laptop (I know I'm bad, calm down) and as long as that's true I won't guarantee 100% uptime. However, I rarely shut down entirely for the night (and will try hard not to when there are users), and only occasionally restart the system to keep it fresh.. so there is some reliability.

In short, If it wasn't obvious, I recommend not using these services in any commercial products, not that I mind (your customers might!). If you want to have some guarantee of uptime, I'm open to inquiries about helping with hosting.

Forums - TODO

The problem with forums is that it requires a lot of state to be done well. For instance, I'd have to add an authentication system so people can build a profile, and not have it compromised through any means, I'd have to do something to protect against spam and bad requests from bots (as someone who doesn't monitor internet traffic, I think you'd be surprised how bad the bot traffic and port scanner situation really is). Along-side all of that, I'd be committing to storing various threads packed full of information. so maybe I ought to hold off on forums until I set up a cloud provider or more disk space \_(o_o)_/.

MultiChat (in development) - allows communication with only a little state information. We simply provide gateways to chat rooms, where you begin seeing typed/streamed information only from the point at which you join the chat. (We only store information about which rooms have been registered for use, and perhaps eventually the number of people/hostname and such)

MultiChat - Download Latest

MultiChat is a console application that interfaces with MI services to register a multicast stream with ease, as well as find and join other multicast data streams that are registered in a public fashion.
