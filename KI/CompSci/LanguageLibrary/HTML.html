HTML - Hyper-Text Markup Language

Language Structure

Information is sorted in a hierarchy using tags of the form <tag_id> and </tag_id> where tag_id specifies the way to interpret and structure the given data (between the start and end tags). All tags have "attributes" that can be set, some tags and their attributes are:

  • <html> - defines scope of hyper-text document instructions
  • <head> - defines head of html document, used to specify meta-information, always specify between html and body start tags. Some tags defined in here are:
    • <meta>: used to give character-encoding, etc.
    • <style>: used to specify CSS styles internally, within the start and end tags
    • <link>: used to link an external file with function specified in rel attribute
      • rel: "stylesheet" - Styles (.css);
  • <body> - defines main body of html document
  • <h1>, <h2>, ... <h6> specifies a heading, where number gives rank
  • <p> defines a paragraph
  • <ul> and <ol> specify unordered (dot bulleted), and ordered (number bulleted) lists
    • <li> specifies a bulleted point of text
  • <a> specifies text that acts as a "link"
    • the attribute href is set equal to the string URL to define the link
  • <img> specifies an image using solely attributes (no end tag)
    • attributes: src = image file, alt = alt text, width, height = pixel dimension (string)
  • <button> specifies a clickable button with overlying text

Styling and CSS

Cascading Style sheets (CSS) is the language and interface through which we specify stylistic properties of a webpage. Although script can be used to specify dynamic styles, it often shares many keywords with CSS to perserve simplicity.

Document Object Model (DOM) and Dynamic HTML with JavaScript

Browsers construct a DOM when a webpage is loaded as a tree of objects. Tree is contructed with elements as nodes, html element as root node, and subtrees created for each "block" within an element.
With the DOM, everything about a webpage can be changed using JavaScript.

DOM Methods and Properties

Methods and Properties of an object are accessed using the dot operator '.', an object in a list/tuple of objects can be accessed with the array indexing operator [index]. Attributes are properties of element objects.

Navigating Elements (using document Object)

  • getElementById(id): returns first element object with specified id
  • getElementsByTagName(name): returns tuple of chronologically ordered elements with specified tagname
  • getElementsByClassName(name): returns tuple of chronologically ordered elements with specified class name
  • createElement(element): creates a new HTML element
  • removeChild(element): removes the specified HTML element
  • appendChild(element): adds an HTML element.
  • replaceChild(newElement, oldElement): replaces an HTML element
  • write(text): writes the text to HTML output stream
