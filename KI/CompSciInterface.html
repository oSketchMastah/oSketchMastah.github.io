Color Coding

In all pages containing language syntax there will be color coding used to help the reader better understand the syntax and its relation to the semantics. Following is the list of color keys that will be used:

  • Type : used to denote a class of entity specified using a type specifier (capitalization is also used)
  • Typed Variadic Paramaters : used to denote that a class of parameter can show up a variable number of times (may be followed by a parameter in curly braces {} as an identifer for the number of parameter used in a following definition
  • Functions/Commands with "Effects" : used to denote functions and commands that have effects on computer
  • Functions/Commands w/o "Effects" : used to denote functions/ commands that do not effect computer/ only returns something
