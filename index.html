This website should serve the purposes of:

  • 1. Providing documentation for MainIntelligence code and APIs.
  • 2. Giving a place for live testing of server code.
  • 3. (Maybe) Providing a couple services to the public.

    • Important: (for users of MI services)

    I'll be honest, I write all the server code myself, and have dealt with enough hostile internet bots to the point where I've decided to take an aggressive defensive policy against bad requests and port scanners. So, to improve the quality of traffic over time, the server is programmed to act as a kind of honeypot program, detecting port scanners and obvious attempts at exploitation. The point is, There will be permanent bans issued without warning If you address the server by IP in a host header, or forget to use a host header in the HTTP request. (ie. use a decent browser/client program) There are other various aspects of the request that are tested, and for which you could be banned (as long as you navigate MI services using this web interface, or MI provided stuff, you are definitely ok). I won't detail too much, but basically if I think you're doing stuff that I don't approve of, it's banworthy.

    You can tell if you were banned if I send a 403 Forbidden response to your HTTP request (this happens the moment the server bans you). From that point on, the server will see your IP prior to reading your request and immediately drop the connection. (To you, it will look like the server no longer exists)

    Please get a hold of me if you think the server made a mistake in banning you. If you are using a cloud provider, I may eventually unban you on my own accord. (or not! it depends on the general reputation of your provider and their client-base)
